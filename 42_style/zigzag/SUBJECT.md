Assignment name  : zigzag
Expected files   : zigzag.c
Allowed functions: write

Description:
Write a program that takes a string and outputs it by alternating the case of
the letters: first uppercase, then lowercase, repeating. Non-alphabetic
characters should be printed as-is and do not affect the alternation.
A newline must be added at the end.

Example:
$> ./zigzag "hello world!"
HeLlO wOrLd!
$> ./zigzag "AaBbCc"
AaBbCc
$> ./zigzag "123 abc DEF!"
123 AbC dEf!
$> ./zigzag
$
